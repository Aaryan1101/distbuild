```bash
java -jar worker/build/libs/worker.jar \

  --coordinator-port 8080 \

```

### After CLI (Simple)
```bash
# Easy to remember, readable
distbuild coordinator start
distbuild worker join

```

## Quick Start Guide

### 1. Initialize Configuration
```bash
cd your-project
java -jar path/to/distbuild.jar init
```
This creates `distbuild.yaml` with your settings.

### 2. Start Coordinator
```bash
distbuild coordinator start
```
Starts coordinator using configuration from `distbuild.yaml`.

### 3. Start Workers
```bash
distbuild worker join
```
Workers automatically discover and connect to coordinator.

### 4. Check Status
```bash
distbuild status
```
Shows all connected workers and system status.

## Configuration File Format

The `distbuild.yaml` generated by `distbuild init`:

```yaml
---
coordinator:
  host: "localhost"
  port: 8080
  cache-dir: "./distbuild-cache"
  discovery-enabled: true
cache:
  dir: "./distbuild-cache"
  ttl-days: 7
  max-size-mb: 1024
workers:
  timeout-seconds: 120
  max-tasks: 4
  java-version: "17"
```

## Installation & Distribution

### Build CLI
```bash
cd C:\Users\jayde\OneDrive\Desktop\Minor Project\distcc
.\gradlew.bat :cli:shadowJar
```
Output: `cli/build/libs/distbuild.jar`

### Install System-Wide (Linux/macOS)
```bash
# Copy JAR to system location
sudo cp cli/build/libs/distbuild.jar /usr/local/lib/

# Create wrapper script
sudo tee /usr/local/bin/distbuild > /dev/null <<EOF
#!/bin/bash
exec java -jar /usr/local/lib/distbuild.jar "\$@"
EOF

# Make executable
sudo chmod +x /usr/local/bin/distbuild

# Test
distbuild --help
```

### Install System-Wide (Windows)
```batch
REM Copy JAR
copy cli\build\libs\distbuild.jar C:\Program Files\distbuild\

REM Create batch script
echo @echo off > C:\Windows\System32\distbuild.bat
echo java -jar "C:\Program Files\distbuild\distbuild.jar" %%* >> C:\Windows\System32\distbuild.bat

REM Test
distbuild --help
```
